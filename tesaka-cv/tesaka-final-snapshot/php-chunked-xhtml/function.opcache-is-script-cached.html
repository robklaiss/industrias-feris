opcache_is_script_cached

(PHP 5 >= 5.5.11, PHP 7, PHP 8, PECL ZendOpcache >= 7.0.4)

opcache_is_script_cached — Tells whether a script is cached in OPCache

Description

opcache_is_script_cached(string $filename): bool

This function checks if a PHP script has been cached in OPCache. This can be used to more easily detect the "warming" of the cache for a particular script. This function only checks in-memory cache, not file cache. In order to check file cache, use opcache_is_script_cached_in_file_cache().

Parameters

filename
The path to the PHP script to be checked.

Return Values

Returns true if filename is cached in OPCache in-memory cache, false otherwise.
