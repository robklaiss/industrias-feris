Requirements

If you have the GD library (available at » http://www.libgd.org/) you will also be able to create and manipulate images.

The format of images you are able to manipulate depend on the version of GD you install, and any other libraries GD might need to access those image formats.

Note: libgd-2.1.0 or higher is required. Alternatively, use the bundled GD library that ships with PHP.

Note: The GD library requires zlib >= 1.2.0.4.

You may wish to enhance GD to handle more image formats.
